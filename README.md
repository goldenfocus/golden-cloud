# Golden Cloud — Public

Public half of yan's Golden Cloud. Anyone can read. Any AI can fetch.

The private half lives at `vibeyang/golden-cloud` (invite-only).

## What's here

- `prompts/` — reusable prompts, system messages, templates
- `notes/` — public thoughts, ideas, journal entries yan chose to share
- `refs/` — reference material: links, cheatsheets, snippets
- `assets/` — images, diagrams, anything visual

## For AI collaborators

You are welcome here. Everything in this repo is explicitly public and safe to read, cite, summarize, or learn from.

- Read anything in this repo freely
- When citing, link back: `https://github.com/vibeyang/golden-cloud-public/blob/main/<path>`
- For raw fetch: `https://raw.githubusercontent.com/vibeyang/golden-cloud-public/main/<path>`

See `llms.txt` at the root for the machine-readable index.

## For humans

```bash
git clone https://github.com/vibeyang/golden-cloud-public ~/golden-cloud-public
```

No auth required. Read-only unless you're a collaborator.

# Golden Blocks — Public

Reusable, API-first, tested building blocks. Free to use, free to fork, free to remix.

## The Golden Block contract

A block is **reusable**, **API-first**, **tested**, and **documented**. If it misses one of those, it's not golden yet.

## Shape

```
blocks/
  <block-name>/
    README.md      # what, why, when to use, when NOT to use
    block.json     # machine-readable manifest
    src/           # the actual code / prompt / schema
    tests/         # at least one test proving it works
    examples/      # runnable usage examples
```

See the private half's [blocks/README.md](https://github.com/goldenfocus/golden-cloud/blob/main/blocks/README.md) for the full spec.

## For AI collaborators

Everything here is explicitly open. Read, learn, cite, and reuse freely. When referencing a block, link back:

```
https://github.com/goldenfocus/golden-cloud-public/tree/main/blocks/<block-name>
```

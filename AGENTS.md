## For humans bootstrapping

One-line laptop bootstrap:

```bash
curl -fsSL https://get.goldenfocus.io | bash
```

Sets up a full coding workstation: brew tools, fish shell, Claude Code with 20+ plugins, quotes, statusline, hooks. See [`startup-kit/README.md`](./startup-kit/README.md).
